# Changelog

All notable changes to this project are documented here.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and the project uses [Semantic Versioning](https://semver.org/).

## Unreleased

## 0.5.0 - 2026-06-23

### Added

- Dynamic NMR overlay filtering for ¹H, ¹³C, and other available nuclei.
- Independent chemical-shift and coupling-constant overlays.
- Pairwise coupling labels in Hz in the molecular view.

## 0.4.0 - 2026-06-22

### Added

- Smooth mouse-wheel and trackpad zoom.
- Yellow 3D highlighting for the selected table atom.

## 0.3.0 - 2026-06-22

### Added

- Arcball rotation with inertia.
- Front, top, side, and reset camera presets.
- Atom picking synchronized with the shielding table.

## 0.2.0 - 2026-06-22

### Added

- Adjustable label size and explicit zoom controls.
- Geometry-based single, double, and triple bond display.

## 0.1.0 - 2026-06-22

### Added

- Initial ORCA output parser.
- Shielding, chemical-shift, and coupling tables.
- TMS reference editing.
- Interactive molecular view.
- TXT and CSV export.
